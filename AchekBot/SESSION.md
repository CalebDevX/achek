# AchekBot Session & WhatsApp Linking

Welcome to the AchekBot session page! Here you can securely link your bot to WhatsApp using the official Baileys method.

## How to Link Your Bot

1. **Start AchekBot**
   - Run the bot using `node index.js`.
   - A QR code will appear in your terminal.

2. **Scan the QR Code**
   - Open WhatsApp > Settings > Linked Devices > Link a Device.
   - Scan the QR code shown in your terminal.

3. **Get Your `creds.json` File**
   - After successful linking, AchekBot will generate a `creds.json` file in the `session` folder.
   - This file contains your WhatsApp session credentials.

4. **Upload/Backup Your `creds.json`**
   - Keep this file safe! You can use it to reconnect your bot without scanning the QR code again.
